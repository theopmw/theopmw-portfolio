About

I’m a developer because I genuinely love learning. The thing I love most about programming is that you’re always a student, there’s always something new to learn! The absolutely massive scope of computer science is something I find incredibly rewarding and exciting and pushes me to continually improve on my skills and develop my knowledge.


In 2020 I took the leap and trained in Software Development at Code Institute. Although challenging, learning programming has to be one of the most rewarding decisions I’ve ever made. Through the course and various self-study, I've gained a solid understanding of computer programming fundamentals and am able to pick up new technologies and tools quickly. My core programming languages are HTML, CSS/Sass, JavaScript and Python and I'm currently investing time in learning React whilst improving my knowledge of cloud-based infrastructure by studying for the AWS Cloud Practitioner exam.


I always aim to build applications that are scalable, maintanable, responsive and accesible whilst providing an engaging UX to ensure users enjoy interacting with the application – on top of that I can even centre a div!


Before getting into tech I worked in hospitality, events and hospitality PR for around 10 years, managing bars, large scale events and PR for a diverse range of clients across London.
